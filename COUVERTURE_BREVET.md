## Épreuves du DNB série générale

Pour les candidats scolaires, le DNB comporte :

- Français ;
- Mathématiques ;
- Histoire-géographie et enseignement moral et civique ;
- Sciences ;
- Épreuve orale.

L'épreuve orale reste volontairement repoussée à la fin du projet, comme prévu dans la roadmap V1.

## Couverture actuelle de la V2

### Français

Couvert :

- grammaire ;
- accords ;
- réécriture ;
- conjugaison ;
- compréhension de texte ;
- figures de style ;
- rédaction.

À renforcer :

- dictée ;
- orthographe lexicale ;
- ponctuation ;
- questions de compréhension longues ;
- rédaction complète avec brouillon, plan et relecture.

### Mathématiques

Couvert :

- nombres relatifs ;
- fractions ;
- puissances ;
- développements ;
- factorisation ;
- équations ;
- arithmétique ;
- proportionnalité ;
- pourcentages ;
- probabilités ;
- statistiques ;
- fonctions ;
- fonctions affines ;
- Pythagore ;
- Thalès ;
- trigonométrie ;
- transformations ;
- volumes ;
- échelles ;
- algorithmique et tableur.

À renforcer :

- calcul mental et automatismes ;
- résolution de problèmes longs ;
- lecture de graphiques et tableaux ;
- grandeurs composées ;
- exercices mixtes type brevet.

### Histoire-Géo EMC

Couvert :

- Seconde Guerre mondiale ;
- Guerre froide ;
- repères historiques ;
- métropoles ;
- espaces productifs ;
- valeurs de la République ;
- analyse de document.

À renforcer :

- Première Guerre mondiale ;
- totalitarismes ;
- décolonisation ;
- construction européenne ;
- Ve République ;
- espaces de faible densité ;
- aménagement du territoire ;
- Union européenne ;
- repères géographiques ;
- développement construit ;
- EMC en situation pratique.

### Sciences

Couvert :

- SVT : effort, organisme, environnement ;
- Physique-chimie : énergie, électricité ;
- Technologie : chaînes d'information et d'énergie ;
- méthode : lire un graphique scientifique.

À renforcer :

- SVT : digestion, reproduction, génétique, immunité, planète Terre ;
- Physique-chimie : mouvement, vitesse, masse volumique, transformations chimiques, pH, signaux ;
- Technologie : programmation, objets connectés, capteurs/actionneurs, matériaux ;
- exercices avec unités et réponses écrites.

Note : pour la session 2026 en métropole, Éduscol indique que l'épreuve de sciences porte sur physique-chimie et SVT en série générale. L'app garde quand même Technologie, car la règle officielle générale reste deux disciplines parmi trois selon les sessions et centres.

## Conclusion

La V2 couvre bien les matières de l'écrit du brevet général, mais elle ne couvre pas encore toute la profondeur attendue.

Priorités de remplissage :

1. Mathématiques : augmenter le nombre d'exercices par cours.
2. Français : ajouter dictée, orthographe et compréhension longue.
3. Histoire-Géo EMC : ajouter les grands chapitres manquants et le développement construit.
4. Sciences : densifier SVT et physique-chimie en priorité, puis technologie.
